# JSnack 2
## L'utente inserisce due parole in successione, con due prompt. Il software stampa prima la parola più corta, poi la parola più lunga.

1. Chiedere tramite prompt la prima parola e inserirla in una variabile.
2. Chiedere tramite prompt la seconda parola e inserirla in una variabile.
3. Determinare quale parola è più corta.
4. Stamparle in ordine di lunghezza.
